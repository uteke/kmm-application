# KMM Application

Presenting a list of Frichti products that redirect on click to the product detail page.

## Technicals
- Gradle Kotlin DSL
- Ktor for networking
- SQLDelight for local storage
- Koin for dependency injection
- Coroutines with flows for data streams
- KotlinX for serialization
- Compose with flows & navigation component for Android
- SwiftUI with Combine & navigation link for iOS

## Architecture
- MVI (Model View Intent) in shared module to centralize presentation logics (can be done in platform side)
- Clean architecture
